# Student Record System 📝

A simple student record management system written in Python.  
It allows users to:

- Add a student
- Delete a student
- Update student name
- View student list

## 🔧 Technologies Used

- Python 3.10+
- `match-case` structure (Python 3.10 feature)

## ▶️ How to Run

1. Make sure you have Python 3.10 or higher installed.
2. Run the file with:

```bash
python student_record.py
